Digit Recognition

Machine Learning Python TensorFlow Computer Vision

Aperçu du Projet

Digit Recognition est une application interactive d'intelligence artificielle qui permet la reconnaissance de chiffres manuscrits en temps réel. L'utilisateur peut dessiner un chiffre sur une interface tactile ou à la souris, et le système prédit instantanément le chiffre dessiné avec un pourcentage de confiance.

Ce projet a été développé pour approfondir mes connaissances en apprentissage automatique et en traitement d'images, tout en créant une application intuitive et pédagogique pour illustrer les concepts de réseaux de neurones convolutifs.

Fonctionnalités Principales

  • Interface de dessin interactive pour saisir des chiffres manuscrits
  • Reconnaissance en temps réel avec indication du niveau de confiance
  • Visualisation des caractéristiques détectées par le réseau neuronal
  • Mode d'apprentissage permettant d'améliorer le modèle avec de nouveaux exemples
  • Dashboard de performance montrant les matrices de confusion et métriques
  • Exportation et importation de modèles entraînés

Détails d'Implémentation

Le cœur du système est un réseau neuronal convolutif (CNN) entraîné sur le dataset MNIST. L'architecture du réseau comprend plusieurs couches de convolution suivies de couches de pooling et finalement des couches denses pour la classification.

Le prétraitement des images dessinées est une étape cruciale: redimensionnement, normalisation et application de filtres pour optimiser la reconnaissance. L'interface utilisateur a été développée avec Tkinter pour assurer une compatibilité sur différentes plateformes.

La fonctionnalité d'apprentissage continu utilise des techniques de fine-tuning et d'augmentation de données pour améliorer les performances du modèle sans nécessiter un réentraînement complet.

Défis et Solutions

Variabilité des styles d'écriture

La grande variabilité dans la façon dont les utilisateurs dessinent les chiffres a posé un défi de généralisation. Pour résoudre ce problème, j'ai implémenté des techniques d'augmentation de données (rotations, translations, dilatations) et intégré un mécanisme permettant d'enrichir le modèle avec les exemples problématiques rencontrés.

Performance sur appareils à faible puissance

Faire fonctionner un modèle de deep learning en temps réel sur des appareils aux ressources limitées était un défi important. J'ai optimisé le modèle par quantification et élagage, et implémenté une version allégée spécifiquement pour les appareils mobiles, réduisant la taille du modèle de 80% tout en maintenant une précision supérieure à 97%.
